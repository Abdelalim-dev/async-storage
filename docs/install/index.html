Installation

#Get library

yarn add @react-native-async-storage/async-storage

#Link

#Android & iOS

  • React Native 0.60+

CLI autolink feature links the module while building the app.

On iOS, use CocoaPods to add the native RNAsyncStorage to your project:

npx pod-install
  • React Native <= 0.59
react-native link @react-native-async-storage/async-storage

#Windows

  • React Native Windows >= 0.63 CLI autolink feature links the module while building the app.

  • React Native Windows <= 0.62 For earlier versions check manual linking

#macOS

For macOS the manual linking is currently the only linking option.
